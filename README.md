# ESP32-S3 Voice Recorder

基础录音存储学习项目：使用 ESP32-S3 读取 INMP441 数字麦克风音频，将录音保存为 WAV 文件到 TF 卡，最后通过电脑读取并播放录音文件。

## 当前目标

第一阶段只实现最基础闭环：

1. ESP32-S3 正常启动并输出串口日志。
2. TF 卡可以创建和读取文件。
3. INMP441 可以采集声音数据。
4. PCM 音频可以封装成 WAV 文件。
5. 电脑可以播放生成的 WAV 文件。

## 暂不做

- 不做 App。
- 不做 Wi-Fi。
- 不做蓝牙。
- 不做 AI 转写。
- 不做屏幕。
- 不做电池。
- 不做 USB 麦克风模式。

## 作品集描述草稿

基于 ESP32-S3 平台预研基础录音存储设备，实现数字麦克风采集、WAV 音频封装、TF 卡文件存储和电脑端文件读取。项目用于学习嵌入式音频采集、文件系统、外设接线和基础产品闭环开发。
