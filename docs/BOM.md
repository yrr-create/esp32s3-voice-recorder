# BOM

## 第一阶段物料

| 物料 | 建议选择 | 数量 | 备注 |
| --- | --- | ---: | --- |
| 主控开发板 | ESP32-S3 N16R8 Type-C 开发板 | 1 | 优先选已焊排针的大板 |
| 麦克风模块 | INMP441 I2S 数字麦克风模块 | 1 | 已焊排针更方便 |
| 存储模块 | TF 卡读写模块，SPI/SDIO，3.3V | 1 | 初期用 SPI 模式 |
| 存储卡 | 8GB 或 16GB microSD/TF 卡 | 1 | 不建议用 128MB 小卡 |
| 连接线 | 杜邦线 | 若干 | 按模块排针选择公母头 |
